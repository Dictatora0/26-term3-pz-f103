# Local Deployment Layout

- `deploy/redis/redis.conf`: Redis service config
- `deploy/elasticsearch/`: Elasticsearch config and heap override
- `deploy/nginx/`: Nginx site configs for iBoot and JetLinks
- `deploy/iboot/config/application-prod.yml`: external iBoot prod config
- `deploy/jetlinks/config/application-prod.yml`: external JetLinks prod config
- `deploy/runtime/`: runtime data, logs, uploaded files, and JetLinks UI assets
- `deploy/OPERATIONS.md`: deployment summary, service info, verification and maintenance commands
