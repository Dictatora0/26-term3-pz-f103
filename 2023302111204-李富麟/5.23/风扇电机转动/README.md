# 普中 STM32F103 风扇电机实验

本工程基于上级目录中的普中 `STM32F103ZE` 标准库模板修改而来，用 `TIM3_CH1(PA6)` 输出 PWM，控制直流风扇电机转动。

## 程序说明

- 入口文件：`User/fan_main.c`
- 风扇驱动：`APP/fan/fan.c`
- PWM 输出口：`PA6`
- PWM 频率：约 `1kHz`
- 占空比范围：`0~1000`

程序下载后会：

1. 初始化串口1、LED、TIM3 PWM
2. 默认以 `70%` 占空比启动风扇
3. 在 `40% -> 55% -> 70% -> 85% -> 100%` 之间循环切换转速
4. 通过板载 LED 指示程序在运行

## 风扇接线

不要把风扇电机直接接到单片机 IO 口。`PA6` 只能输出控制信号，必须通过驱动级带动风扇。

### 推荐接法：N 沟道 MOS 管低端驱动

需要器件：

- 1 个直流风扇电机
- 1 个 N 沟道逻辑级 MOS 管，例如 `AO3400`、`IRLZ44N`
- 1 个续流二极管，例如 `1N5819`、`1N4007`
- 1 个 `220Ω` 栅极电阻
- 1 个 `10kΩ` 栅极下拉电阻
- 独立风扇电源，按风扇额定电压供电，例如 `5V` 或 `12V`

接线如下：

- `PA6` -> `220Ω电阻` -> `MOS管G极`
- `G极` -> `10kΩ电阻` -> `GND`
- `MOS管S极` -> 开发板 `GND`
- `MOS管D极` -> 风扇负极
- 风扇正极 -> 外部电源正极
- 外部电源负极 -> 开发板 `GND`
- 续流二极管并在风扇两端：
  - 二极管负极接风扇正极
  - 二极管正极接风扇负极

## 如果你手里是小功率 5V 风扇

如果是常见的小型 `5V` 两线风扇，也按上面方式接，不要直接接 `PA6`。

可以这样供电：

- 风扇正极接开发板 `5V`
- 风扇负极接 MOS 管 `D`
- 开发板 `GND` 和 MOS 管 `S`、风扇电源负极共地

前提是：

- 风扇工作电流不能超过开发板 `5V` 供电能力
- USB 供电足够稳定

## 调速说明

如果风扇不容易启动，可以把 `User/fan_main.c` 里的默认启动占空比调高：

- `FAN_START_DUTY 700U` 可改成 `850U` 或 `1000U`

如果你想让风扇一直全速转动，可以把循环改成固定：

```c
FAN_SetSpeed(1000);
while (1)
{
}
```
